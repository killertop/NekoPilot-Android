# 贡献指南 / Contributing guide

欢迎提交 issue 和改进建议。请先搜索现有 issue，并使用可复现的步骤、设备/Android 版本、节点协议类型和已脱敏日志说明问题。不要在 issue、截图或日志中提交订阅链接、节点凭据、密钥或个人信息。
Issues and improvements are welcome. Search existing issues first and include reproducible steps, device/Android version, protocol type, and sanitized logs. Never include subscription links, node credentials, keys, or personal data in issues, screenshots, or logs.

提交代码前，请保持改动聚焦，运行与改动相关的测试和 Lint，并说明用户可见影响。涉及 VPN、规则、DNS、导入或订阅更新的改动需要设备回归。
Before contributing code, keep changes focused, run relevant tests and lint, and describe user-visible impact. Changes to VPN, rules, DNS, import, or subscription refresh require device regression.
